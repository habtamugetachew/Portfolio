## ⚙️ Local Setup

To run this portfolio locally:

```bash
git clone https://github.com/habtamugetachew/Portfolio.git
cd Portfolio
npm install
npm run dev
```

Then open the local Vite URL shown in the terminal (typically `http://localhost:5173`).
